# 采购任务概述

## 功能说明

根据采购需求调研结果，自动分析并生成采购任务概述，包含采购分类、方式推荐、审批层级、组织架构、材料清单等关键决策。

## 前置条件

必须基于 `procurement-requirements` 的调研结果执行。如调研结果缺失核心信息（项目名称、预算金额、采购类别、需求部门），先引导用户补充。

## 执行流程

### 第一步：采购分类判断

根据调研结果中的预算金额和集中采购目录情况，判断采购类型：

| 类型 | 条件 |
|------|------|
| **集中采购** | 纳入集中采购目录 **且** 单次采购金额 ≥ 50万元 |
| **分散采购** | 未纳入目录 **或** 金额 < 50万元 |
| **IT委托采购** | 委托集团下属共享信息技术服务的非保险子公司开展的IT类采购（第四十二条） |
| **联合采购** | 集团公司统一组织的联合采购（第四十一条） |

> 单次采购金额确定规则（第五十三条）：固定金额 → 发生额上限 → 预算金额 → 预估发生额

### 第二步：采购方式推荐

根据采购类型和金额范围推荐：

#### 分散采购

| 金额范围 | 采购方式 | 依据 |
|----------|----------|------|
| < 20万元 | 自行采购（质优价廉、便利高效原则） | 第二十六条 |
| 20万（含）- 50万元 | 参照集中采购方式选择 | 第二十五条 |

#### 集中采购（≥ 50万元）

根据项目特征匹配采购方式，优先级从高到低：

1. **公开招标** — 默认优先，法律法规要求必须招标的强制执行
2. **邀请招标** — 有限范围供应商、招标费用过大、时间紧急（非法定必须招标）
3. **竞争性谈判** — 招标失败、技术复杂不能确定规格、时间紧急、不能事先计算价格
4. **竞争性磋商** — 只能提功能性指标/宽泛规格、技术复杂、科研项目、工程建设项目
5. **询价** — 需求客观明确、规格标准统一、现货充足且价格变化小
6. **单一来源** — 唯一供应商、紧急情况、项目一致性、涉密（需加强论证+支撑材料，情形一/三需外网公示3个工作日）

### 第三步：审批权限分析

根据采购类型和金额确定审批路径：

#### 立项审批

| 金额 | 审批层级 |
|------|----------|
| < 20万元（分散） | 无需单独审批，与合同/财务审批合并 |
| 20万-50万元（分散） | 财务负责人会签 → 分管/协管领导审批 |
| 50万-100万元（集中） | 报集采委审议 |
| ≥ 100万元（集中） | 集采委审议 → 总经理室审批 |

#### 特殊审批

- **三重一大/党委会前置/超出董事会授权** → 提交党委会研究讨论
- **外部数据采购** → 数据安全自评估报告报大数据服务部、信息技术部、法律合规部审批
- **计划外集中采购** → 集采委审批

### 第四步：组织架构要求

根据采购类型确定需要成立的组织：

| 组织 | 20万以下 | 20万-50万分散 | ≥50万集中 | 公开招标 |
|------|----------|---------------|-----------|----------|
| 评审小组 | 无需 | ≥3人 | ≥5人（含法律合规） | ≥7人（外部专家≥2/3） |
| 项目工作组 | 需求部门组建 | 需求部门组建 | 财务+需求部门组建 | 财务+需求部门组建 |
| 验收小组 | 2人以上（非经办人） | 同左 | 同左 | 同左 |

### 第五步：材料清单生成

根据采购类型、金额、特殊情形，动态生成所需材料清单。以下为各阶段通用要件：

#### 立项阶段
- 项目立项申请（含项目名称、类别、需求部门、预算金额、采购方式、是否涉密、是否外部数据）
- 必要性/可行性论证材料
- 市场/同业调研报告
- 潜在供应商情况说明
- 候选供应商基本信息

#### 实施阶段
- 采购方案请示（含需求、方式、评审小组构成、评审方法、邀请供应商名单）
- 采购文件（含合同主要条款）
- 法律合规部门审核意见

#### 结果阶段
- 采购结果申请
- 供应商ESG行为准则（所有供应商）

> 根据特殊情形追加材料：涉密项目追加保密责任状、外部数据追加安全自评估报告、中介服务追加信息安全风险评估报告、单一来源追加公示材料等。

## 注意事项

- 采购前公示（单一来源情形一/三需外网公示3个工作日）和采购后信息公开（第三十五条）是两个不同要求，不可混淆
- 采购评审小组应独立于采购项目工作组，不得为采购办及纪委监察人员
- 验收人员不得为采购实施经办人
- 不得将应当公开招标的项目化整为零规避公开招标
