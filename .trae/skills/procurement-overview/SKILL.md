# 采购任务概述

## 功能说明

根据《中国人寿再保险有限责任公司采购管理办法》（中再寿险发〔2026〕21号）规定，结合采购需求调研信息，自动分析并生成采购任务概述，包含采购分类判断、方式推荐、审批层级、组织架构、材料清单等内容，覆盖全流程关键决策。

## 触发场景

- 完成采购需求调研后
- 用户需要确认采购计划
- 用户询问"生成采购任务概述"、"汇总采购信息"等

## 适用范围确认（第五条、第二条）

在生成概述前，应确认采购项目属于本办法管理范围：
- **属于**：工程类、货物类、服务类采购
- **不属于**：日常业务招待、食品、自办会务和临时所需的小额购买等（第五条排除项）
- **境外采购**：应按照当地相关法律法规，参照本办法执行（第二条）

如确认不属于本办法管理范围，应告知用户无需按本办法流程执行。

## 核心功能

### 一、采购分类判断

根据《办法》规定，采购项目的基本分类为集中采购与分散采购，另有两种特殊采购情形：

#### 1. 集中采购（第三条）
同时满足以下两个条件：
- 纳入集中采购目录（附件1）
- 单次采购金额 ≥ 50万元

#### 2. 分散采购
不满足集中采购任一条件的，即为分散采购：
- 金额方面：单次采购金额 < 50万元
- 或目录方面：未纳入集中采购目录

#### 3. IT 委托采购（第四十二条）
属于委托集团下属共享信息技术服务的非保险子公司开展的 IT 类采购：
- 在公司与受托方签署的委托管理协议项下
- 信息技术部门统筹组织受托方按照公司采购管理办法履行公司内部审批程序后
- 由受托方负责执行相关具体工作

#### 4. 联合采购（第四十一条）
属于集团公司统一组织的联合采购：
- 由集团公司相关部门按照集团公司采购管理办法进行采购
- 过程中涉及需公司审批的事项，按公司相关审批权限进行

### 二、单次采购金额确定规则（第五十三条）

| 情形 | 金额确定方式 |
|------|--------------|
| 固定金额采购 | 以该固定金额确定 |
| 无法确定最终发生额但有明确上限 | 以该发生额上限确定 |
| 既无固定金额也无明确上限 | 以该采购行为的预算金额确定 |
| 以上方法均无法适用 | 以经批准的预估发生额确定 |

**重要提示**：单次采购是指对同一供应商的同一标的采购行为

### 三、采购方式推荐

#### 3.1 集中采购项目（≥50万元）

##### 招标方式

| 情形 | 推荐方式 | 依据条款 | 关键备注 |
|------|----------|----------|----------|
| 纳入集中采购范围，原则上优先 | 公开招标或邀请招标 | 第十八条 | 审慎选择单一来源采购方式 |
| 法律法规或监管规定要求必须公开招标 | 公开招标 | 第十九条第一款 | 强制性要求 |
| 总经理室认为需要公开招标 | 公开招标 | 第十九条第二款 | - |
| 具有特殊性，只能从有限范围供应商处采购 | 邀请招标 | 第二十条第一款 | 不少于3家 |
| 公开招标费用占项目总价值比例过大 | 邀请招标 | 第二十条第二款 | - |
| 公开招标时间难以满足紧急需要（非法定必须招标） | 邀请招标 | 第二十条第三款 | 法定必须招标的除外 |
| 总经理室认为需要邀请招标 | 邀请招标 | 第二十条第四款 | - |

##### 非招标方式

| 情形 | 推荐方式 | 依据条款 | 关键备注 |
|------|----------|----------|----------|
| 招标后无供应商投标/无合格标的/重新招标未成立 | 竞争性谈判 | 第二十一条第一款 | - |
| 技术复杂或性质特殊，不能确定详细规格 | 竞争性谈判 | 第二十一条第二款 | 市场上仅2家的应在采购方案中说明 |
| 招标时间不能满足紧急需要 | 竞争性谈判 | 第二十一条第三款 | - |
| 不能事先计算出价格总额 | 竞争性谈判 | 第二十一条第四款 | - |
| 总经理室认为需要竞争性谈判 | 竞争性谈判 | 第二十一条第五款 | - |
| 只能提出功能性指标或宽泛技术规格 | 竞争性磋商 | 第二十二条第一款 | 如服务项目 |
| 技术复杂或性质特殊，不能确定详细规格 | 竞争性磋商 | 第二十二条第二款 | - |
| 因专利、专有技术或服务时间数量不能确定价格总额 | 竞争性磋商 | 第二十二条第三款 | - |
| 市场竞争不充分的科研项目 | 竞争性磋商 | 第二十二条第四款 | - |
| 依法必须招标工程以外的工程建设项目 | 竞争性磋商 | 第二十二条第五款 | - |
| 总经理室认为需要竞争性磋商 | 竞争性磋商 | 第二十二条第六款 | - |
| 需求客观明确，规格标准统一，现货充足且价格变化小 | 询价 | 第二十三条第一款 | 不少于3家供应商报价比较 |
| 有独特行业销售模式（如非大批量公务车采购） | 询价 | 第二十三条第二款 | - |
| 总经理室认为需要询价 | 询价 | 第二十三条第三款 | - |

##### 单一来源采购（第二十四条）

| 情形 | 特殊要求 | 依据 |
|------|----------|------|
| 只能从唯一供应商处采购（专利、专有技术、国家机关合作等） | 加强立项论证 + 加大调研力度 + 支撑材料 + 外网公示3个工作日 | 第二十四条第一款 |
| 发生不可预见紧急情况不能从其他供应商采购 | 加强立项论证 + 加大调研力度 + 支撑材料 | 第二十四条第二款 |
| 必须保证原有项目一致性或配套要求 | 加强立项论证 + 加大调研力度 + 支撑材料 + 外网公示3个工作日 | 第二十四条第三款 |
| 涉及国家秘密或企业秘密不宜采用其他方式 | 加强立项论证 + 加大调研力度 + 支撑材料 | 第二十四条第四款 |
| 总经理室认为需要单一来源 | 加强立项论证 + 加大调研力度 + 支撑材料 | 第二十四条第五款 |

**单一来源特别强调**：
- 必须满足上述适用情形，同时加强立项论证、加大市场及同业调研力度，并提供支撑性材料
- 属于情形（一）（三）的，立项获批后还需在公司外网公示3个工作日（不含当日）
- 公示信息包括：项目名称、采购内容、供应商名称、服务期限等

#### 3.2 分散采购项目（<50万元）

| 金额范围 | 采购方式要求 | 依据条款 |
|----------|--------------|----------|
| 20万元（不含）以下 | 自行采购，按照质优价廉、便利高效原则（如直接报价比较或说明） | 第二十六条 |
| 20万元（含）- 50万元（不含） | 参照集中采购方式要求选择合适的采购方式 | 第二十五条 |

| 维度 | 公开招标 | 邀请招标 | 竞争性谈判 | 竞争性磋商 | 询价 | 单一来源 |
|------|----------|----------|------------|------------|------|----------|
| 供应商数 | 不限 | ≥3家 | ≥3家（仅2家需说明） | ≥3家（仅2家需说明） | ≥3家 | 1家 |
| 确定方式 | 综合评分 | 综合评分 | 最低价 | 基准价+综合得分 | 最低价 | 谈判 |
| 招标代理 | 必须委托 | 必须委托 | 可自办或委托 | 可自办或委托 | 可自办或委托 | 自办 |
| 采购前公示 | 否 | 否 | 否 | 否 | 否 | 情形（一）（三）需外网公示3个工作日 |
| 采购后信息公开 | 是（集中采购） | 是（集中采购） | 是（集中采购） | 是（集中采购） | 是（集中采购） | 是（集中采购，涉密除外） |

**注意**：采购前公示（第二十四条）和采购后信息公开（第三十五条）是两个不同的制度要求，不可混淆。

### 四、审批权限分析

#### 4.1 项目立项审批（第三十七条）

| 金额范围 | 采购类型 | 发起方 | 审批层级 |
|----------|----------|--------|----------|
| <20万元 | 分散采购 | 需求部门 | 无需单独审批，与合同审批或财务审批合并（第三十六条） |
| 20万元（含）- 50万元（不含） | 分散采购 | 需求部门 | 财务管理部门负责人会签 → 需求部门及归口管理部门（如有）的分管/协管领导审批。采购方案与项目立项合并报批 |
| 50万元（含）- 100万元（不含） | 集中采购 | 需求部门 | 报集采委审议 |
| ≥100万元 | 集中采购 | 需求部门 | 报集采委审议 → 报总经理室审批 |

#### 4.2 采购方案审批（第三十八条）

| 采购类型 | 发起方 | 审批流程 |
|----------|--------|----------|
| 分散采购 | 需求部门 | 采购方案与项目立项合并报批 |
| 集中采购 | 财务管理部门 | 会签需求部门及归口管理部门（如有）→ 报集采委审批 |

#### 4.3 采购结果审批（第三十九条）

| 金额范围 | 采购类型 | 发起方 | 审批层级 |
|----------|----------|--------|----------|
| 20万元（含）- 50万元（不含） | 分散采购 | 需求部门 | 财务管理部门负责人会签 → 需求部门及归口管理部门（如有）的分管/协管领导审批。合同签署可与采购结果合并报批 |
| ≥50万元 | 集中采购 | 财务管理部门 | 报集采委审议 |

#### 4.4 特殊审批路径

| 情形 | 审批要求 | 依据条款 |
|------|----------|----------|
| 计划外集中采购 | 需经集采委审批后方可执行 | 第九条 |
| 涉及"三重一大"事项 | 需提交党委会研究讨论 | 第三十七条第四款 |
| 属于党委会前置研究事项清单的重大项目 | 需提交党委会研究讨论 | 第三十七条第四款 |
| 超出董事会授权范围的采购事项 | 根据公司章程及授权方案规定履行审批 | 第三十七条第四款 |
| **涉及外部数据采购** | **需求部门就数据安全自评估报告报大数据服务部、信息技术部和法律合规部审批** | **第三十九条第三款** |
| 涉及托管银行、投资银行服务等 | 如集团公司或公司另有规定的从其规定 | 第四条 |

#### 4.5 审批方式（第四十条）

- 总经理室和集采委可通过OA系统或会议形式对采购事项进行审批
- 分散采购项目：采购立项同采购方案、采购结果同合同签署的审批可以合并进行
- **合并审批时，审批节点应同时满足采购各环节相关审批规定**

### 五、组织架构要求

#### 5.1 各机构职责速查

| 机构 | 主要职责 |
|------|----------|
| 总经理室 | 审批采购制度；审议年度预算；按权限审批集中采购事项（第十一条） |
| 集采委 | 审议采购制度；确定目录及限额；审批集中采购计划；按权限审批集中采购事项（第十二条） |
| 采购办（设在财务管理部门） | 起草制度；拟定目录；组织编制计划；负责集中采购实施；协助需求部门；管理监督员库（第十三条） |
| 采购项目工作组 | 负责采购项目实施（第十四条） |
| 采购评审小组 | 资质审查、评标/谈判/磋商、出具评审意见（第十四条） |
| 需求部门 | 编制计划和预算；组织分散采购/参与集中采购；签署合同；验收及付款；信息公开；归档（第十五条） |
| 财务管理部门 | 预算管理；履行采购办职责（第十六条） |
| 法律合规部门 | 审核合同等法律文件；协助制定采购文件范本（第十六条） |
| 纪检部门 | 监督采购活动；处理违规违纪行为（第十六条） |
| 办公室 | 通过公司网站披露采购信息（第十六条） |

#### 5.2 采购项目工作组（第十四条）

| 项目类型 | 组建方式 |
|----------|----------|
| 集中采购 | 财务管理部门会同需求部门组建 |
| 分散采购 | 需求部门负责组建 |

#### 5.3 采购评审小组（第十四条）

| 项目类型 | 成员人数 | 特殊要求 |
|----------|----------|----------|
| 20万元以下采购 | 无需成立评审小组 | - |
| 分散采购（≥20万元） | ≥3人 | 含1名需求部门人员 |
| 集中采购（≥20万元） | ≥5人 | 含1名需求部门人员 + 1名法律合规部门人员 |
| 单一来源采购（≥20万元） | 按对应类型 | 含1名法律合规部门人员 |
| 公开招标 | ≥7人 | 外部评审专家 ≥ 2/3，成员为采购项目工作组以外人员 |
| 邀请招标 | 依法组成评标委员会 | - |

**核心原则**：
- 采购评审小组应独立于采购项目工作组
- 不得为采购办及纪委监察人员
- 应有1名需求部门人员
- 集中采购和20万元以上的单一来源采购还应包含1名法律合规部门人员

#### 5.4 验收小组（第三十四条）

- 两人以上组成
- 验收人员不得为采购实施经办人
- 依据采购文件、投标文件、采购合同等对交付成果进行验收
- 验收结论须由全体参与验收人员共同签字确认
- 验收形成的纸质资料应予以留存，提交付款时作为附件上传

#### 5.5 集中采购监督员（第五十一条）

- 公司以委派集中采购监督员的方式对集中采购活动进行监督
- 详见附件4《集中采购监督员管理规则》
- 监督员对存在违规行为应予以纠正；拒不纠正或可能造成损失的，应责令停止采购

### 六、供应商管理要求

#### 6.1 供应商准入条件（第四十四条）

供应商需满足以下基本条件：
1. 依法注册成立并具有独立运作能力（运营、财务、组织、管理、市场）
2. 可在相应国家和地区提供产品和服务，且符合有关国家或行业标准
3. 有意愿和能力为公司提供产品和服务，并接受公司监督
4. 满足公司供应商ESG行为准则中环境管理、社会责任、企业管治方面的要求
5. 没有公司不可接受的不良记录
6. 其他应具备的条件

#### 6.2 供应商ESG要求细则

- 采购实施环节，采购文件中应要求潜在供应商签署《ESG行为准则》
- 潜在供应商提供的应答文件中应做出响应（第四十四条第四款）
- 所有供应商均需提供《供应商环境、社会和企业管治（ESG）行为准则》（第三十三条第四款）

#### 6.3 供应商日常管理（第四十五条）

需求部门负责对正在合作的供应商进行日常管理：
- 创建和维护供应商的基本信息
- 持续跟踪其行业资质变化
- 监控重大舆情情况
- 关注监管处罚信息
- 跟踪诚信记录变化
- 及时向财务管理部门报备

### 七、材料清单生成

#### 7.1 采购计划阶段（第八条、第九条）

需求部门提出采购计划应包括：
- [ ] 采购项目名称
- [ ] 采购需求描述
- [ ] 采购预算
- [ ] 拟提交采购立项时间
- [ ] 采购完成时点要求
- [ ] 项目联系人

集中采购计划由财务管理部门汇总报集采委审批；分散采购计划由财务管理部门留存备案。

#### 7.2 项目立项阶段（第二十九条）

**必需要件**：
- [ ] 项目立项申请
  - 项目名称、项目类别、需求单位、需求部门
  - 预算金额、采购方式
  - 是否涉密、是否涉及核心经营信息情况
  - 是否涉及外部数据采购
- [ ] 项目必要性论证材料
- [ ] 项目可行性论证材料
- [ ] 市场调研情况报告
- [ ] 同业调研情况报告
- [ ] 潜在供应商情况说明
- [ ] 拟邀请供应商选择的条件和原因、基本情况（公开招标除外）
- [ ] 候选供应商基本信息（第四十四条准入条件）

**按需要件**：
- [ ] 财务管理部门建议的采购方式、拟邀请供应商条件及基本情况（集中采购项目）
- [ ] 中介机构信息安全风险评估报告（涉及核心经营信息的中介服务项目）
- [ ] 保密责任状（涉及核心经营信息的中介服务项目）
- [ ] 单一来源公示材料（单一来源采购情形一、三，公示3个工作日）

#### 7.3 采购实施阶段（第三十一条、第三十二条）

**必需要件**：
- [ ] 采购方案请示
  - 明确采购需求
  - 采购方式
  - 采购评审小组构成
  - 评审方法
  - 拟邀请供应商名单（公开招标除外）
- [ ] 采购文件（含合同主要条款）
- [ ] 法律合规部门审核意见（采购文件合同主要条款、采购合同）

**按需要件**：
- [ ] 市场上仅有2家供应商的情况说明（竞争性谈判/磋商且市场仅2家时）
- [ ] 委托招标代理机构协议（招标类集中采购）
- [ ] ESG行为准则签署要求（采购文件中应要求供应商签署）

#### 7.4 采购结果阶段（第三十三条）

**必需要件**：
- [ ] 采购结果申请
  - 项目基本情况（同立项）
  - 供应商背景和资质材料
  - 收费标准、价格比选情况
  - 评审情况及结果建议（评审小组构成、评审过程、采购结果）
- [ ] 《供应商环境、社会和企业管治（ESG）行为准则》（所有供应商均需提供）

**按需要件**：
- [ ] 中介机构及人员信息安全风险评估报告（涉及核心经营信息的中介服务项目）
- [ ] 数据安全自评估报告（涉及外部数据采购）

#### 7.5 合同签署与验收阶段（第三十四条）

- [ ] 采购合同（集中采购项目必须签订）
- [ ] 验收报告（合同约定需要验收的项目）
  - 两人以上验收小组签字确认
  - 验收人员不得为采购实施经办人
  - 依据采购文件、投标文件、采购合同对交付成果进行验收
  - 验收形成的纸质资料应予以留存，提交付款时作为附件上传

#### 7.6 信息公开阶段（第三十五条）

**集中采购项目信息公开**：
- [ ] 合同审批环节就披露工作进行说明（所有集中采购项目的通用要求）
- [ ] 成交结果公告（合同签订之日起3个工作日内，不含当日）
  - 采购内容
  - 采购方式
  - 候选供应商
  - 中选供应商
  - 合同确定的采购数量
  - 采购价格等基本要素
- [ ] 公开信息披露申请及审批流程（需求部门发起，第十五条第四款）

**例外情况**：
- 涉及国家秘密、商业秘密的采购事项可不披露
- 构成国家秘密、商业秘密并拟不进行披露的，应当在合同审批环节说明理由

#### 7.7 采购后评价阶段（第四十六条）

**供应商后评价**（20万元以上各类采购）：
- [ ] 对供应商在合同签署、执行、验收、投入使用等表现进行综合评价
- [ ] 供应商等级评定（优秀、合格、不合格）
- [ ] 此结论作为后续管理该供应商承担同类业务的基本依据

**使用情况后评价**（20万元以上资讯服务类、系统类、软件类等）：
- [ ] 使用效果综合评价
- [ ] 使用频次统计
- [ ] 未使用原因分析
- [ ] 此结论作为后续研究此类采购必要性的基本依据

**注意**：
- 采购后评价由财务管理部门统一组织，需求部门配合
- 后评价人员与实施人员应相互分离
- 财务管理部门根据后评价结果反馈对供应商的改进意见和后续使用建议

#### 7.8 采购档案管理（第四十七条、第四十八条）

- [ ] 项目立项审批资料
- [ ] 采购请示审批资料
- [ ] 采购结果审批资料
- [ ] 采购实施过程相关资料原件
- [ ] 采购相关会议材料
- [ ] 采购合同等资料原件

**归档要求**：
- 需求部门应对采购过程中的所有相关资料予以整理并归档
- 应当每年定期完成上一年度档案的移交工作
- 由公司档案管理部门集中统一保管
- 符合公司档案管理有关规定

### 八、采购纪律与监督检查

#### 8.1 回避制度（第四十九条）

公司采购活动实行回避制度：
- 采购相关人员与投标人（或供应商）有利害关系的，应当回避
- 与供应商主要负责人有亲属关系的，应当回避
- 投标人（或供应商）认为采购相关人员与其他供应商有利害关系的，可以申请其回避
- 采购相关人员包括：集采委成员、采购办人员、需求部门负责人及采购参与人员

#### 8.2 禁止行为清单（第五十条）

在采购过程中，禁止以下行为：
1. 互相串通损害公司利益、国家利益、社会公共利益和其他当事人的合法权益
2. **将符合集中采购条件的项目化整为零或通过其他方式违反集中采购程序和方式**
3. 对供应商实行差别待遇或者歧视待遇
4. 接受供应商的贿赂或者从供应商获取其他不正当利益
5. 正式开标前即泄露标底
6. 在评定的中后期，改变采购标准、中标候选人顺序
7. 在中标候选人确定后改变采购产品、采购标准，或采购未中标产品
8. 隐匿、销毁应当保存的采购文件或者伪造、变造采购文件
9. 以作为或不作为的方式阻碍或拒绝监督
10. 其他实质性影响到采购公平、公正、公开的行为

#### 8.3 监督机制

- 集中采购监督员：对集中采购活动进行监督（第五十一条）
- 纪检部门：对采购活动进行监督和检查（第五十二条）
- 违规处理：严重违反规定造成较大损失的，依据相关规定追究责任；违纪违规的按有关规定处理；涉嫌犯罪的移交司法机关

### 九、采购流程概览

```
采购计划编制 → 项目立项 → 采购实施 → 采购结果 → 合同签署 → 信息公开 → 采购后评价 → 档案管理
     ↓            ↓          ↓          ↓          ↓          ↓          ↓          ↓
  年度/季度    审批权限    组建工作组   评审    法律审核   成交公告   供应商评价  资料归档
  预算管理    按金额分级   制定方案    呈报结果  签署合同   3个工作日内  使用情况评价  年度移交
                          组建评审小组          验收（如需）
                          组织评审
```

**分场景流程**：

- **IT委托采购（第四十二条）**：信息技术部门统筹 → 受托方按采购管理办法 → 公司内部审批 → 受托方执行
- **联合采购（第四十一条）**：集团公司按集团管理办法 → 公司权限内审批事项按公司权限
- **计划外采购（第九条）**：需求部门提出 → 集采委审批 → 方可执行

## 注意事项

1. 生成的概述仅供参考，最终以实际审批结果为准
2. 用户可修改任何内容，系统会重新分析相关影响
3. 涉及"三重一大"范围的项目需提交党委会研究讨论（第三十七条第四款）
4. 超出董事会授权范围的采购事项需根据公司章程及授权方案履行审批（第三十七条第四款）
5. 采购评审小组应独立于采购项目工作组，不得为采购办及纪委监察人员（第十四条）
6. 验收人员不得为采购实施经办人（第三十四条）
7. 集中采购项目必须进行信息公开，涉密除外（第三十五条）
8. 不得将应当公开招标的项目化整为零规避公开招标（第十八条）
9. 计划外集中采购项目需经集采委审批后方可执行（第九条）
10. 联合采购由集团公司按集团采购管理办法执行，涉及公司审批事项按公司权限进行（第四十一条）
11. IT委托采购由信息技术部门统筹履行公司内部审批程序（第四十二条）
12. 后评价人员与实施人员应相互分离（第四十六条）
13. 采购档案每年定期完成上一年度移交，由档案管理部门集中统一保管（第四十七条）
14. 实行回避制度，采购相关人员与供应商有利害关系的应当回避（第四十九条）
15. 集中采购监督员对违规行为应纠正，拒不纠正或可能造成损失的应责令停止采购（第五十一条）
16. 外部数据采购需报大数据服务部、信息技术部和法律合规部审批（第三十九条第三款）
17. 所有供应商均需提供ESG行为准则（第三十三条第四款）
18. 合并审批时，审批节点应同时满足采购各环节相关审批规定（第四十条）
19. 未经授权不得与供应商签署或订立任何口头或书面形式的协议、合同、书据等或做出有法律约束力的单方承诺（第三十四条）
20. 本办法不适用于日常业务招待、食品、自办会务和临时所需的小额购买等（第五条）
