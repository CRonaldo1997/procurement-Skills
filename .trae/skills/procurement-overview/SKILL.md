# 采购任务概述

## 功能说明

根据《中国人寿再保险有限责任公司采购管理办法》规定，结合采购需求调研信息，自动分析并生成采购任务概述，包含采购分类判断、方式推荐、审批层级、材料清单等内容。

## 触发场景

- 完成采购需求调研后
- 用户需要确认采购计划
- 用户询问"生成采购任务概述"、"汇总采购信息"等

## 核心功能

### 采购分类判断
根据预算金额自动判断：
- **分散采购**：< 50万元
- **集中采购**：≥ 50万元

### 采购方式推荐
| 条件 | 推荐方式 |
|------|----------|
| 金额≥50万 | 公开招标 |
| 供应商有限 | 邀请招标 |
| 技术复杂 | 竞争性谈判/磋商 |
| 标准化需求 | 询价 |
| 仅1家供应商 | 单一来源采购 |

### 审批权限分析
| 金额范围 | 审批层级 |
|----------|----------|
| < 20万元 | 无需单独审批 |
| 20-50万元 | 分管领导审批 |
| 50-100万元 | 集采委审议 |
| ≥ 100万元 | 集采委+总经理室审批 |

### 材料清单生成
- **必需要件**：项目立项申请、采购方案请示、供应商资质、ESG确认函
- **按需要件**：信息安全评估报告（涉及核心信息）、数据安全自评估报告（涉及外部数据）、单一来源公示材料等

## 输出示例

```markdown
# 采购任务概述

## 一、基本信息
| 项目 | 内容 |
|------|------|
| 项目名称 | 2026年度办公设备采购项目 |
| 采购类别 | 货物类 |
| 预算金额 | 80万元 |
| 采购分类 | 集中采购 |
| 需求部门 | 行政部 |

## 二、采购方式分析
- **推荐方式**：公开招标
- **推荐理由**：依据办法第18条，金额≥50万元应采用公开招标

## 三、审批权限
| 环节 | 审批层级 |
|------|----------|
| 项目立项 | 集采委审议 |
| 采购方案 | 集采委审议 |
| 采购结果 | 集采委审议→总经理室审批 |

## 四、所需材料清单
- [ ] 项目立项申请
- [ ] 采购方案请示
- [ ] 供应商资质材料
- [ ] ESG行为准则确认函

## 五、下一步操作
- [ ] 确认概述信息
- [ ] 开始撰写材料
```

## 交互流程

接收调研结果 → 分析采购分类 → 推荐采购方式 → 确定审批权限 → 生成材料清单 → 输出概述文档 → 用户确认/修改

## 注意事项

1. 生成的概述仅供参考，最终以实际审批结果为准
2. 用户可修改任何内容，系统会重新分析相关影响
3. 涉及"三重一大"范围的项目需提交党委会研究
