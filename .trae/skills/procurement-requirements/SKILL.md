# 采购需求调研

## 功能说明

根据《中国人寿再保险有限责任公司采购管理办法》（中再寿险发〔2026〕21号）要求，通过结构化问答方式全面收集采购项目信息，确保后续材料撰写的准确性和完整性。调研内容涵盖材料编制所需的所有关键决策要素。

## 触发场景

- 开始采购工作前的信息收集
- 需要撰写采购立项申请材料
- 用户询问"采购需求调研"、"收集采购信息"等

## 适用范围确认（第五条）

在开始调研前，应首先确认采购项目是否属于本办法管理范围：
- **属于**：工程类、货物类、服务类采购
- **不属于**：日常业务招待、食品、自办会务和临时所需的小额购买等（第五条排除项）
- **境外采购**：公司在境外开展的采购活动应按照当地相关法律法规，参照本办法执行（第二条）

如确认不属于本办法管理范围，应告知用户无需按本办法流程执行。

## 调研流程

### 一、基本信息
1. **项目名称**：请提供采购项目的正式名称
2. **采购类别**：工程类 / 货物类 / 服务类
   - 工程类：包括房地产购置、自建、装修等
   - 货物类：包括房屋配套设备、信息化设备等
   - 服务类：各类专业服务采购（如中介服务、技术服务等）
3. **预算金额**：预计采购金额（万元）
4. **金额类型**（第五十三条，决定单次采购金额的计算口径）：
   - 固定金额：以该固定金额确定单次采购金额
   - 有明确发生额上限：以该发生额上限确定单次采购金额
   - 无固定金额也无明确上限：以预算金额确定单次采购金额
   - 预估发生额：以上方法均无法适用时，以经批准的预估发生额确定
5. **是否纳入集中采购目录**：是 / 否 / 不确定
   - 集中采购须同时满足两个条件：纳入目录且金额≥50万元（第三条）
6. **是否属于集团公司联合采购范围**：是 / 否
   - 联合采购由集团公司按集团采购管理办法执行，涉及公司审批事项按公司权限进行（第四十一条）

### 二、组织信息
7. **需求部门**：提出采购需求的具体部门
8. **归口管理部门**：如有，请提供（部分项目可能涉及归口管理）
9. **项目联系人**：负责本项目采购工作的联系人及联系方式
10. **分管领导**：需求部门及归口管理部门的分管/协管领导（第三十七条）

### 三、预算与计划（第八条、第九条）
11. **是否已纳入年度预算**：是 / 否
    - 第八条：各需求部门/归口管理部门应在每年编制次年年度费用及资本性支出预算时，统筹考虑下年度计划采购项目预算，经分管领导确认后，报财务管理部门汇总，按照年度预算规定程序批准后生效
12. **是否已纳入季度预算**：是 / 否
    - 每季度末编制费用预算时，统筹考虑下季度需支付的采购项目预算（第八条）
13. **是否属于计划外采购**：是 / 否
    - 计划外集中采购项目需经集采委审批后方可执行（第九条）
14. **年度预算编制情况**：已按年度预算程序编制 / 由财务管理部门在年度预算内调剂
15. **季度预算编制情况**：已按季度预算程序编制 / 待编制
16. **拟提交采购立项时间**：如"2026年7月"（第九条采购计划要素）
17. **采购完成时点要求**：如"2026年12月31日前"（第九条采购计划要素）

### 四、IT 采购特殊情形（第四十二条）
18. **是否属于委托集团下属共享信息技术服务的非保险子公司开展的 IT 类采购**：是 / 否
    - 如是，在公司与受托方签署的委托管理协议项下，信息技术部门统筹组织受托方按照公司采购管理办法履行公司内部审批程序后，由受托方负责执行相关具体工作

### 五、采购方式与供应商
19. **是否有指定采购方式偏好**：如有，请说明原因
20. **潜在供应商数量预估**：≥3家 / 2家 / 仅1家
    - 如≥3家：可选公开招标、邀请招标、竞争性谈判、竞争性磋商、询价
    - 如仅2家：竞争性谈判/磋商需在采购方案中说明（第二十一条、第二十二条）
    - 如仅1家：可能适用单一来源采购（第二十四条）
21. **是否涉及委托招标代理机构**：是 / 否
    - 招标类集中采购需委托招标代理机构执行，严格落实"管采分离""采需分离"（第二十七条）
22. **是否涉及委托采购代理机构（非招标类）**：是 / 否
    - 非招标类采购可自行采购或委托外部代理机构办理（第二十七条）

### 六、中介服务专项（第二十八条第二款、第二十九条第七款）
23. **是否涉及中介服务采购**：是 / 否
24. **是否涉及核心经营信息**：是 / 否
    - 如涉及，采购过程中的信息安全管理应严格按照中介机构信息安全相关管理规定执行（第二十八条第二款）
    - 如涉及，需准备中介机构信息安全风险评估报告及保密责任状（第二十九条第七款）

### 七、安全保密
25. **是否涉及国家秘密**：是 / 否
    - 如涉及，适用单一来源采购（第二十四条第四款），且可不进行信息公开披露（第三十五条）
26. **是否涉及商业秘密**：是 / 否
    - 如涉及，可不进行信息公开披露，构成商业秘密拟不披露的，应在合同审批环节说明理由（第三十五条）
    - 如涉及，亦可能适用单一来源采购（第二十四条第四款："涉及国家秘密或企业秘密不适宜进行竞争性采购的项目"）
27. **是否涉及外部数据采购**：是 / 否
    - 如涉及，需准备数据安全自评估报告，并报大数据服务部、信息技术部和法律合规部审批（第三十九条第三款）
    - 采购结果阶段还需提供外部数据采购相关的数据安全自评估报告（第三十三条第五款）

### 八、时间与紧急程度
28. **预计完成时间**：如"2026年6月30日前"
29. **是否属于紧急采购**：是 / 否
    - 紧急采购可能影响采购方式选择（邀请招标第二十条第三款、竞争性谈判第二十一条第三款）
30. **是否属于不可预见的紧急情况**：是 / 否
    - 如属于"公司经营发生了不可预见的紧急情况，不能采用其他方式且只能从某特定供应商采购"，可能适用单一来源采购（第二十四条第二款）

### 九、需求描述
31. **采购需求详情**：
    - 具体内容和技术要求
    - 数量、规格、标准
    - 交付要求和时间计划
    - 验收标准（如有）
32. **采购需求是否客观明确**：是 / 否
    - 如只能提出功能性指标或宽泛技术规格，可能适用竞争性磋商（第二十二条第一款）
33. **标的物是否为成品，规格标准统一**：是 / 否
    - 如是且现货充足、价格变化幅度小，可能适用询价（第二十三条第一款）

### 十、前期准备工作（第二十九条立项申请要素）
34. **项目必要性论证情况**：已完成 / 待完成 / 需协助
35. **项目可行性论证情况**：已完成 / 待完成 / 需协助
36. **市场调研情况**：已完成 / 待完成 / 需协助
37. **同业调研情况**：已完成 / 待完成 / 需协助
38. **潜在供应商情况了解程度**：充分 / 一般 / 不足
39. **候选供应商基本信息是否已准备**：是 / 否
    - 第四十四条：需求部门或归口管理部门提出立项申请时，应提供候选供应商基本信息
    - 供应商准入条件包括：依法注册、符合标准、接受监督、满足ESG要求、无不良记录

### 十一、特殊情形
40. **是否涉及"三重一大"事项**：是 / 否
    - 如涉及，需提交党委会研究讨论（第三十七条第四款）
41. **是否属于党委会前置研究事项**：是 / 否
    - 属于党委会前置研究事项清单的重大项目，需提交党委会研究讨论（第三十七条第四款）
42. **是否超出董事会授权范围**：是 / 否
    - 如超出，需根据公司章程及授权方案规定履行相应审批（第三十七条第四款）
43. **是否涉及节能环保、绿色低碳产品**：是 / 否
    - 公司应优先采购此类产品（第六条）
44. **是否涉及专利、专有技术**：是 / 否
    - 如涉及，可能适用单一来源采购（第二十四条第一款）
45. **是否为保证原有项目一致性需再次采购**：是 / 否
    - 如涉及，可能适用单一来源采购（第二十四条第三款）
    - 如适用此情形，立项获批后需在公司外网公示3个工作日（不含当日）
46. **是否涉及与国家机关、企事业单位、科研院所等开展技术/数据/服务合作**：是 / 否
    - 如涉及，可能适用单一来源采购（第二十四条第一款）
47. **是否涉及托管银行、投资银行服务、证券经纪商等**：是 / 否
    - 如集团公司或公司另有规定的从其规定（第四条）
48. **采购相关人员是否与潜在供应商存在利害关系**：是 / 否 / 不确定
    - 公司采购活动实行回避制度，有利害关系的应当回避（第四十九条）
    - 采购相关人员包括：集采委成员、采购办人员、需求部门负责人及采购参与人员

## 交互特点

- 每次提问1-2个问题，避免信息过载
- 支持回答确认和修改
- 不确定的问题可选择"暂不确定"跳过
- 动态调整问题顺序，优先收集关键信息
- 对涉及保密信息的问题进行特别提示
- 根据已回答问题智能推断后续问题
- 对"是"的回答自动展开追问细节

## 注意事项

1. 调研结果将作为后续材料撰写的基础，请尽量提供准确信息
2. 涉及敏感信息的问题将严格保密处理
3. 关键信息缺失可能影响后续采购方式推荐和材料完整性
4. 根据《办法》第二十九条，立项申请需包含项目基本情况、必要性可行性论证、市场调研、采购方式选择等要素
5. 涉及核心经营信息的中介服务项目，需严格按照中介机构信息安全相关管理规定执行（第二十八条第二款）
6. 单次采购是指对同一供应商的同一标的采购行为（第五十三条）
7. 供应商准入需满足依法注册、符合行业标准、接受监督、符合ESG准则、无不良记录等条件（第四十四条）
8. 采购评审小组应独立于采购项目工作组，不得为采购办及纪委监察人员（第十四条）
9. 验收人员不得为采购实施经办人（第三十四条）
10. 不得将应当公开招标的项目化整为零规避公开招标（第十八条）
11. 实行回避制度：采购相关人员与供应商有利害关系的应当回避（第四十九条）
12. IT委托采购由信息技术部门统筹组织受托方按公司采购管理办法履行审批程序（第四十二条）
13. 未经授权不得与供应商签署或订立任何口头或书面形式的协议、合同、书据等或做出有法律约束力的单方承诺（第三十四条）
