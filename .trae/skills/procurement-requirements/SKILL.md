# 采购需求调研

## 功能说明

根据《中国人寿再保险有限责任公司采购管理办法》（中再寿险发〔2026〕21号）要求，通过结构化问答方式全面收集采购项目信息，确保后续材料撰写的准确性和完整性。

## 触发场景

- 开始采购工作前的信息收集
- 需要撰写采购立项申请材料
- 用户询问"采购需求调研"、"收集采购信息"等

## 调研流程

### 一、基本信息
1. **项目名称**：请提供采购项目的正式名称
2. **采购类别**：工程类 / 货物类 / 服务类
3. **预算金额**：预计采购金额（万元）
4. **金额类型**：固定金额 / 框架协议（需提供上限）

### 二、采购方式
5. **采购方式偏好**：是否有指定采购方式
6. **潜在供应商数量**：≥3家 / 2家 / 仅1家

### 三、组织信息
7. **需求部门**：提出采购需求的具体部门
8. **归口管理部门**：如有，请提供

### 四、安全保密
9. **涉及保密信息**：国家秘密 / 商业秘密 / 无
10. **涉及核心经营信息**：是 / 否
11. **涉及外部数据采购**：是 / 否

### 五、时间需求
12. **预计完成时间**：如"2026年6月30日前"
13. **是否紧急**：是 / 否

### 六、需求描述
14. **采购需求详情**：具体内容、技术要求、数量、交付要求等

### 七、前期工作
15. **必要性论证**：已完成 / 待完成 / 需协助
16. **市场调研**：已完成 / 待完成 / 需协助

## 交互特点

- 每次提问1-2个问题，避免信息过载
- 支持回答确认和修改
- 不确定的问题可选择"暂不确定"跳过
- 动态调整问题顺序，优先收集关键信息

## 输出示例

```markdown
# 采购需求调研结果

## 基本信息
- 项目名称：2026年度办公设备采购项目
- 采购类别：货物类
- 预算金额：80万元
- 采购分类：集中采购

## 组织信息
- 需求部门：行政部
- 归口管理部门：无

## 安全与保密
- 涉及保密信息：无
- 涉及核心经营信息：否
- 涉及外部数据采购：否

## 下一步建议
建议调用 procurement-overview 生成采购任务概述
```

## 注意事项

1. 调研结果将作为后续材料撰写的基础，请尽量提供准确信息
2. 涉及敏感信息的问题将严格保密处理
3. 关键信息缺失可能影响后续采购方式推荐和材料完整性
