# 采购需求调研

## 功能说明

根据《中国人寿再保险有限责任公司采购管理办法》（中再寿险发〔2026〕21号），通过结构化问答收集采购项目信息。调研结果将直接用于后续的采购任务概述（procurement-overview）和采购文件撰写（procurement-document-writer）。

## 适用范围确认

在开始调研前，先确认采购项目是否属于本办法管理范围：
- **属于**：工程类、货物类、服务类采购
- **不属于**：日常业务招待、食品、自办会务和临时所需的小额购买等（第五条排除项）
- **境外采购**：应按照当地相关法律法规，参照本办法执行（第二条）

如不属于管理范围，告知用户无需按本办法流程执行。

## 交互原则

- **每次提问1-2个问题**，避免信息过载
- **优先收集关键信息**：项目名称、预算金额、采购类别、需求部门
- **支持跳过**：用户可回答"暂不确定"，后续再补充
- **智能推断**：根据已回答内容自动调整后续问题，避免重复询问
- **对"是"的回答自动追问细节**：如涉及中介服务、涉密、外部数据等，需展开追问
- **涉密信息特别提示**：涉及国家秘密或商业秘密的问题，提醒用户注意保密

## 调研流程

调研分为**核心信息**和**补充信息**两个阶段。核心信息是后续流程必须的，补充信息根据项目情况按需收集。

### 第一阶段：核心信息（必须收集）

这些是判断采购方式、审批权限的基础，**必须全部获取**：

1. **项目名称**
2. **采购类别**：工程类 / 货物类 / 服务类
3. **预算金额**（万元）
4. **金额类型**（第五十三条）：固定金额 / 有明确上限 / 无固定金额也无上限 / 预估发生额
5. **是否纳入集中采购目录**：是 / 否 / 不确定
6. **需求部门**
7. **项目联系人及联系方式**
8. **采购需求详情**：具体内容、技术要求、数量规格、交付要求、验收标准

### 第二阶段：补充信息（按需收集）

根据第一阶段的答案，判断需要追问哪些补充信息：

#### 预算与计划
- 是否已纳入年度/季度预算
- 是否属于计划外采购
- 拟提交立项时间、采购完成时点要求

#### 采购方式与供应商
- 潜在供应商数量预估：≥3家 / 2家 / 仅1家
- 是否有指定采购方式偏好
- 是否涉及委托招标/采购代理机构

#### 特殊情形判断
根据项目特征，判断是否涉及以下情形并追问：

| 情形 | 触发条件 | 追问内容 |
|------|----------|----------|
| IT委托采购 | 涉及IT系统/技术服务 | 是否属于委托集团下属共享信息技术服务的非保险子公司 |
| 中介服务 | 涉及中介服务采购 | 是否涉及核心经营信息 |
| 涉密采购 | 涉及敏感数据/技术 | 是否涉及国家秘密或商业秘密 |
| 外部数据采购 | 涉及数据采购 | 需准备数据安全自评估报告 |
| 紧急采购 | 时间紧迫 | 是否属于不可预见的紧急情况 |
| 单一来源 | 供应商仅1家或专利/专有技术 | 是否涉及专利、专有技术、国家机关合作、保证项目一致性 |
| 三重一大 | 重大项目 | 是否涉及"三重一大"、党委会前置研究、超出董事会授权 |
| 节能环保 | 涉及产品采购 | 是否涉及节能环保、绿色低碳产品 |
| 利害关系 | 任何采购 | 采购相关人员是否与潜在供应商存在利害关系 |

#### 前期准备工作
- 项目必要性/可行性论证情况
- 市场/同业调研情况
- 候选供应商基本信息是否已准备

## 注意事项

- 调研结果将作为后续采购方式推荐和材料撰写的基础，请用户尽量提供准确信息
- 单次采购是指对同一供应商的同一标的采购行为（第五十三条）
- 不得将应当公开招标的项目化整为零规避公开招标（第十八条）
- 未经授权不得与供应商签署或订立任何口头或书面形式的协议（第三十四条）
